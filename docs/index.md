# Helix

**The open-source modular AI home assistant — the Linux PC of smart speakers.**

Helix is a fully modular, open-source AI home assistant built on Raspberry Pi 5. Every hardware component is swappable, every software layer is programmable, and all AI inference runs locally on-device.

## Status

🚧 **In active development.** Follow progress on [GitHub](https://github.com/attunelabs/helix-os) and [Discord](https://discord.gg/MCMYB8aA).
